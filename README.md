### TicTacToe

Простая реализация игры «Крестики-нолики» для двух игроков на C++. Игроки по очереди ставят свои символы (x – первый игрок, o – второй) на поле 3×3. Побеждает тот, кто первым выстроит три своих знака в ряд.



## Требования

- C++17



## Компиляция и запуск

Скомпилируйте программу любым компилятором с поддержкой C++17:

```bash
g++ -std=c++17 -o tic_tac_toe tic_tac_toe.cpp
```

И запустите:

```bash
./tic_tac_toe
```
